# 第 1 週今日 Python 練習題：個人資料收集器

## 題目說明

你要建立一個簡單的「個人資料收集程式」，透過與使用者互動，收集姓名、年齡與身高，再根據輸入資料進行判斷並格式化輸出。

---

## 技術要求

- **概念**：變數、資料型態（str、int、float）、`print`、`input`、型態轉換（`int()`、`float()`）、`if/elif/else` 條件判斷
- **需要的模組**：無（純內建語法）

---

## 具體任務

1. 使用 `input()` 分別詢問使用者的：
   - 姓名（字串）
   - 年齡（整數）
   - 身高（浮點數，單位：公分）
2. 將年齡轉換為 `int`，身高轉換為 `float`
3. 根據年齡，用 `if/elif/else` 判斷並輸出對應的人生階段：
   - 年齡 < 13：「兒童」
   - 13 ≤ 年齡 < 18：「青少年」
   - 18 ≤ 年齡 < 65：「成年人」
   - 年齡 ≥ 65：「樂齡族」
4. 根據身高，判斷：
   - 身高 < 160：「矮個子」
   - 160 ≤ 身高 < 175：「標準身高」
   - 身高 ≥ 175：「高個子」
5. 最後用 `print` 輸出完整的個人資料摘要

---

## 範例

```python
# 執行後互動如下：

請輸入您的姓名：小明
請輸入您的年齡：16
請輸入您的身高（公分）：168.5

# 輸出結果：
===== 個人資料摘要 =====
姓名：小明
年齡：16 歲（青少年）
身高：168.5 公分（標準身高）
========================
```

---

## 提示

- `input()` 回傳的值永遠是字串，記得用 `int()` 或 `float()` 轉換
- 字串可以用 f-string 格式化：`f"姓名：{name}"`
- `if`、`elif`、`else` 的縮排要一致（4 個空格）
- 可以先把「人生階段」存進一個變數，再用 f-string 輸出

---

## 學習重點

這道題雖然簡單，卻涵蓋了往後爬蟲與資料分析中最核心的基礎：

- **資料型態轉換**：爬蟲抓回來的資料幾乎都是字串，需要轉成數字才能計算
- **條件判斷**：資料清洗時常需要根據欄位值分類處理（例如：價格 > 1000 才儲存）
- **f-string**：輸出結果到 CSV 或印出報告時非常常用

---

## 進階挑戰（選做）

1. **輸入驗證**：加入錯誤處理，當使用者輸入非數字時，顯示「請輸入有效的數字」並重新詢問（提示：用 `try/except` 或先判斷 `str.isdigit()`）
2. **BMI 計算**：額外詢問體重（公斤），計算 BMI（體重 / (身高/100)²），並根據 BMI 值輸出健康狀態（過輕 < 18.5、正常 18.5–24.9、過重 25–29.9、肥胖 ≥ 30）
3. **多人版本**：用 `while` 迴圈（下週主題預告！）讓程式可以連續輸入多人資料，輸入姓名為「quit」時結束
