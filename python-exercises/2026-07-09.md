# 第 1 週今日 Python 練習題：BMI 健康計算器

## 題目說明

建立一個 BMI（身體質量指數）計算器，讓使用者輸入身高與體重，程式計算出 BMI 值並判斷健康狀態，最後輸出個人化的建議訊息。

這題會綜合運用本週所學的變數、資料型態轉換、輸入輸出與條件判斷。

## 技術要求
- 概念：變數、資料型態（`int`、`float`、`str`）、`print`、`input`、`if/elif/else`、算術運算子
- 需要的模組：無（純 Python 內建即可）

## 具體任務

1. 使用 `input()` 向使用者詢問**姓名**、**身高（公分）** 和 **體重（公斤）**
2. 將輸入的數字字串轉換為適當的數值型態（`float`）
3. 依公式計算 BMI：`BMI = 體重(kg) / (身高(m))²`
   - 注意：身高需從公分換算為公尺（÷ 100）
4. 使用 `if/elif/else` 判斷 BMI 所屬範圍並給出分類：

   | BMI 範圍 | 分類 |
   |----------|------|
   | < 18.5 | 體重過輕 |
   | 18.5 ~ 24.9 | 正常體重 |
   | 25.0 ~ 29.9 | 體重過重 |
   | ≥ 30.0 | 肥胖 |

5. 最後用 `print()` 輸出一段包含姓名、BMI 數值（四捨五入到小數點後兩位）及健康建議的個人化訊息

## 範例

```python
# 輸入
請輸入您的姓名：小明
請輸入您的身高（公分）：175
請輸入您的體重（公斤）：80

# 輸出
小明 您好！
您的 BMI 為：26.12
健康狀態：體重過重
建議：適量運動並控制飲食，目標 BMI 維持在 18.5 ~ 24.9 之間。
```

## 提示

- `input()` 回傳的永遠是**字串**，記得用 `float()` 轉換數字
- 身高從公分轉公尺：`height_m = height_cm / 100`
- 次方運算可以用 `**`：`height_m ** 2`
- 四捨五入用 `round(數值, 小數位數)`，例如 `round(bmi, 2)`
- 條件判斷的順序很重要，先判斷最小值或最大值再往中間收斂

## 學習重點

**與爬蟲／ML 的關聯：**
- **資料型態轉換**是爬蟲與資料分析的核心技能——從網頁抓下來的數字永遠是字串，必須自己轉換
- **條件判斷**在 ML 中用於資料清洗時過濾異常值（例如排除不合理的 BMI < 5 或 > 100）
- 熟悉 `input/print` 流程有助於之後撰寫互動式資料收集腳本

## 進階挑戰（選做）

1. **輸入驗證**：如果使用者輸入的身高或體重不是有效數字（例如輸入文字），用 `try/except` 捕捉錯誤並提示重新輸入
2. **多人模式**：用 `while True` 讓程式持續詢問下一位使用者，直到輸入 `q` 才結束
3. **理想體重計算**：根據身高反推 BMI 22（正常範圍中間值）對應的理想體重，一起輸出給使用者參考
   - 理想體重公式：`ideal_weight = 22 × (height_m)²`
