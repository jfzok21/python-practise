# 第 1 週今日 Python 練習題：BMI 計算器與體重分類

## 題目說明

撰寫一個 BMI（身體質量指數）計算器。程式會請使用者輸入姓名、身高（公分）與體重（公斤），計算出 BMI 值後，依照世界衛生組織（WHO）的標準輸出對應的體重分類結果。

## 技術要求
- 概念：變數、資料型態（int / float / str）、print、input、if/elif/else、型態轉換（int()、float()）
- 需要的模組：無（純內建語法）

## 具體任務
1. 使用 `input()` 取得使用者的姓名、身高（cm）和體重（kg）
2. 將身高與體重轉換成適當的數值型態（float）
3. 依公式計算 BMI：`BMI = 體重(kg) / (身高(m) ** 2)`（注意：身高要先從公分換算成公尺）
4. 使用 `if/elif/else` 依下列標準判斷體重分類：
   - BMI < 18.5 → 體重過輕
   - 18.5 ≤ BMI < 24.0 → 正常體重
   - 24.0 ≤ BMI < 27.0 → 體重過重
   - 27.0 ≤ BMI < 30.0 → 輕度肥胖
   - 30.0 ≤ BMI < 35.0 → 中度肥胖
   - BMI ≥ 35.0 → 重度肥胖
5. 格式化輸出結果，BMI 值保留小數點後兩位

## 範例
```python
# 輸入
請輸入您的姓名：小明
請輸入您的身高（公分）：175
請輸入您的體重（公斤）：70

# 輸出
----------------------------------------
小明 您好！
身高：175.0 cm　體重：70.0 kg
您的 BMI 值為：22.86
體重狀態：正常體重
----------------------------------------
```

## 提示
- `input()` 回傳的永遠是字串（str），記得用 `float()` 轉換數值
- 身高換算：公尺 = 公分 / 100，例如 `height_m = height_cm / 100`
- 次方運算子是 `**`，例如 `2 ** 2` 等於 4
- 格式化輸出可用 f-string：`f"{bmi:.2f}"` 會保留兩位小數
- `print("-" * 40)` 可以快速印出分隔線

## 學習重點

**與後續爬蟲／ML 的關聯：**
- **資料型態轉換**：爬蟲抓回來的網頁資料全都是字串，必須轉換成數值才能計算，這和 `float(input(...))` 的概念完全一樣
