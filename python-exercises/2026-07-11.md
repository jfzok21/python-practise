# 第 1 週今日 Python 練習題：簡易四則運算計算機

## 題目說明

設計一個互動式的「四則運算計算機」程式。程式啟動後，詢問使用者輸入兩個數字及一個運算符號（`+`、`-`、`*`、`/`），然後計算並輸出結果。過程中要處理各種邊界情況，例如除以零、輸入非數字等，讓程式更健壯（robust）。

這道題會深化你對 `input()`、型態轉換（`int`、`float`）、`if/elif/else` 多分支判斷的理解，也是 Python 「取得使用者輸入 → 處理資料 → 輸出結果」這個最基本流程的完整練習。

---

## 技術要求

- 概念：變數、資料型態（`int`、`float`、`str`）、`input()`、型態轉換、`if/elif/else`、`print()` 格式化輸出
- 需要的模組：無（純 Python 內建語法即可）

---

## 具體任務

1. 使用 `input()` 取得使用者輸入的**第一個數字**，並用 `float()` 轉換為浮點數。
2. 使用 `input()` 取得使用者輸入的**運算符號**（`+`、`-`、`*`、`/`）。
3. 使用 `input()` 取得使用者輸入的**第二個數字**，並用 `float()` 轉換為浮點數。
4. 用 `if/elif/else` 判斷運算符號，執行對應的計算：
   - `+`：加法
   - `-`：減法
   - `*`：乘法
   - `/`：除法（需額外判斷除數是否為 `0`，若為 `0` 則輸出錯誤提示）
5. 如果使用者輸入了不支援的運算符號（例如 `%`、`^`），輸出「不支援的運算符號！」。
6. 輸出結果時，若結果為整數（例如 `6.0`），請以整數形式顯示（`6`）；若有小數則顯示到小數點後兩位。

---

## 範例

```python
# --- 範例一：正常加法 ---
# 輸入：
#   第一個數字：10
#   運算符號：+
#   第二個數字：5
# 輸出：
#   10 + 5 = 15

# --- 範例二：除以零 ---
# 輸入：
#   第一個數字：8
#   運算符號：/
#   第二個數字：0
# 輸出：
#   錯誤：除數不能為 0！

# --- 範例三：除法有小數 ---
# 輸入：
#   第一個數字：10
#   運算符號：/
#   第二個數字：3
# 輸出：
#   10 / 3 = 3.33

# --- 範例四：不支援的符號 ---
# 輸入：
#   第一個數字：5
#   運算符號：^
#   第二個數字：2
# 輸出：
#   不支援的運算符號：^
```

---

## 提示

- `float("10")` 可以把字串 `"10"` 轉換成浮點數 `10.0`；整數輸入也可以用 `float()` 處理，不必另外判斷。
- 用 `result % 1 == 0` 可以判斷一個浮點數是否為整數（例如 `6.0 % 1 == 0` 為 `True`）。
- 格式化輸出可以用 f-string：`f"{a} {op} {b} = {result:.2f}"`，或在判斷是否整數後分別輸出。
- 判斷除以零：在 `/` 的分支裡，先用 `if b == 0:` 處理，再做除法。

## 學習重點

這道題是「使用者輸入 → 型態轉換 → 條件判斷 → 格式化輸出」的完整鏈條。  
往後學爬蟲時，你會大量接觸「取得資料（抓網頁）→ 處理資料（解析 HTML）→ 輸出結果（存 CSV）」，這個三段式思維現在就開始建立。  
學 ML 時，`if/elif/else` 的邏輯也會以決策樹（Decision Tree）的形式再次出現。

## 進階挑戰（選做）

1. **迴圈模式**：加上 `while True` 迴圈，讓程式計算完一次後詢問「是否繼續？（y/n）」，輸入 `n` 才結束。
2. **錯誤處理**：用 `try/except ValueError` 包住 `float()` 轉換，讓使用者輸入非數字時（例如 `abc`）不會崩潰，改為輸出友善的錯誤訊息並重新詢問。
3. **歷史記錄**：用一個 `list` 儲存每次計算的算式與結果，結束時一次列印所有歷史紀錄。
