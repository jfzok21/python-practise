# 第 1 週今日 Python 練習題：BMI 健康計算機

## 題目說明

製作一個簡單的 BMI（身體質量指數）計算機。程式會向使用者詢問身高與體重，計算出 BMI 值，並根據結果判斷健康狀態。這個練習涵蓋了變數、資料型態轉換、print、input 以及 if/elif/else 的核心概念。

## 技術要求
- 概念：變數、資料型態（`int`、`float`、`str`）、`print`、`input`、`if/elif/else`、算術運算
- 需要的模組：無（純 Python 內建即可）

## 具體任務

1. 使用 `input()` 請使用者輸入身高（公分）與體重（公斤），並將字串轉換為適當的數值型態。
2. 將身高從公分轉換為公尺，再用公式 `BMI = 體重 / 身高²` 計算 BMI 值。
3. 使用 `if/elif/else` 判斷 BMI 落在哪個區間，並輸出對應的健康狀態：
   - BMI < 18.5：體重過輕
   - 18.5 ≤ BMI < 24：正常體重
   - 24 ≤ BMI < 27：過重
   - 27 ≤ BMI < 30：輕度肥胖
   - BMI ≥ 30：中重度肥胖
4. 最後以格式化字串（f-string）輸出結果，BMI 值保留兩位小數。

## 範例

```python
# 輸入
請輸入您的身高（公分）：170
請輸入您的體重（公斤）：65

# 輸出
您的 BMI 值為：22.49
健康狀態：正常體重 ✅
```

```python
# 輸入
請輸入您的身高（公分）：160
請輸入您的體重（公斤）：80

# 輸出
您的 BMI 值為：31.25
健康狀態：中重度肥胖 ⚠️
```

## 提示

- `input()` 回傳的永遠是字串，記得用 `float()` 轉換數值，才能做計算。
- 身高從公分轉公尺：`height_m = height_cm / 100`
- 計算平方可以用 `**` 運算子：`height_m ** 2`
- f-string 格式化小數位數：`f"{bmi:.2f}"` 會保留兩位小數。
- 判斷區間時，Python 支援連續比較：`18.5 <= bmi < 24` 是合法語法。
