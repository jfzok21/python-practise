# 第 1 週今日 Python 練習題：簡易成績分析器

## 題目說明

你要建立一個「學生成績分析程式」。程式依序詢問學生姓名與五科成績，接著計算總分、平均分、最高分與最低分，最後根據平均分判斷等第，輸出一張格式整齊的成績單。

這道題整合了本週所有核心概念：變數宣告、資料型態轉換、多重條件判斷、f-string 格式化輸出，是第一週很好的收尾練習。

---

## 技術要求

- **概念**：變數、資料型態（`str`、`int`、`float`）、`input`、`print`、型態轉換（`int()`）、`if/elif/else`、基本算術運算子、內建函式 `max()` / `min()`
- **需要的模組**：無（純內建語法）

---

## 具體任務

1. 使用 `input()` 詢問學生姓名（字串）
2. 分別使用 `input()` 詢問五科成績（國文、英文、數學、自然、社會），用 `int()` 轉換為整數
3. 計算：
   - **總分**：五科相加
   - **平均分**：總分除以 5（用 `round()` 四捨五入到小數點後一位）
   - **最高分**：用 `max()` 函式
   - **最低分**：用 `min()` 函式
4. 根據平均分，用 `if/elif/else` 判斷等第：
   - 平均 ≥ 90：「優（A）」
   - 80 ≤ 平均 < 90：「良（B）」
   - 70 ≤ 平均 < 80：「可（C）」
   - 60 ≤ 平均 < 70：「差（D）」
   - 平均 < 60：「不及格（F）」
5. 用 `print` 與 f-string 輸出格式整齊的成績單（參考範例）

## 範例

```python
# 執行後互動如下（使用者輸入以「>」標示）

請輸入學生姓名：> 小華

英文成績：> 78
數學成績：> 92
自然成績：> 67
社會成績：> 80

# 輸出結果：
╔══════════════════════════╗
║      學生成績單          ║
╠══════════════════════════╣
║ 姓名：小華               ║
║ 國文：85 分              ║
║ 英文：78 分              ║
║ 數學：92 分              ║
║ 自然：67 分              ║
║ 社會：80 分              ║
╠══════════════════════════╣
║ 總分：402 分             ║
║ 平均：80.4 分            ║
║ 最高：92 分              ║

╚══════════════════════════╝
```

（提示：邊框可以用普通的 `print("=" * 30)` 代替，不一定要畫框線）

---

## 提示

- `int(input("國文成績："))` 可以一行完成「顯示提示 → 讀取輸入 → 轉換為整數」
- `max(ch, en, math, sci, soc)` 可以直接找出五個變數中的最大值
- `round(total / 5, 1)` 可以計算平均並四捨五入到小數點後一位
- f-string 範例：`f"║ 姓名：{name}"` — 花括號內可以放任何變數或運算式
- 等第判斷要注意順序：先從最高分開始判斷，否則條件可能永遠不會被觸發

---

## 學習重點

這道題直接對應未來的資料處理技能：

- **網頁爬蟲**：從網頁抓下來的資料（例如商品評分、股價）幾乎都是字串，需要先 `int()` 或 `float()` 轉換才能計算統計值，`max()` / `min()` 是最常用的基礎函式
- **機器學習**：分類問題的本質就是「根據數值決定類別」——就像你今天用平均分判斷等第一樣；`if/elif/else` 是手寫規則分類器（rule-based classifier）的基礎形式，之後的 `sklearn` 決策樹會自動學習這些規則
