# 第 2 週今日 Python 練習題：九九乘法表與數字統計

## 題目說明

今天的練習將深入運用 Python 的迴圈語法，透過兩個任務讓你熟悉 `for`、`while`、`range`、`break` 和 `continue` 的實際應用。

---

## 技術要求

- 概念：`for` 迴圈、`while` 迴圈、`range()`、`break`、`continue`、巢狀迴圈
- 需要的模組：無（純 Python）

---

## 具體任務

### 任務一：九九乘法表

1. 使用巢狀 `for` 迴圈，印出 1 到 9 的九九乘法表。
2. 每列格式對齊，讓輸出整齊易讀。
3. 跳過所有結果為偶數的算式（使用 `continue`）。

### 任務二：猜數字遊戲

1. 使用 `while` 迴圈，讓使用者不斷猜一個預設的秘密數字（如 42）。
2. 每次猜錯時提示「太大了」或「太小了」。
3. 猜對時用 `break` 跳出迴圈，並顯示共猜了幾次。
4. 限制最多猜 10 次，超過則顯示「遊戲結束，正確答案是 [數字]」。

### 任務三：數字統計

1. 使用 `for` 迴圈搭配 `range(1, 101)`，找出 1 到 100 中所有能被 3 整除但不能被 5 整除的數字。
2. 印出這些數字，並在最後顯示總計個數與總和。

---

## 範例

```python
# 任務一 輸出範例（部分）
# 1x1=1   （奇數，印出）
# 1x2=2   （偶數，跳過）
# 1x3=3   （奇數，印出）
# ...

# 任務二 輸出範例
# 請猜一個 1~100 的數字：50
# 太大了！
# 請猜一個 1~100 的數字：25
# 太小了！
# 請猜一個 1~100 的數字：42
# 恭喜你！共猜了 3 次！

# 任務三 輸出範例
# 3 6 9 12 18 21 24 27 33 ...
# 符合條件的數字共 47 個，總和為 2418
```

---

## 提示

- `range(start, stop, step)` 可以控制起點、終點和步進值。
- 巢狀迴圈中，內層的 `break` 只會跳出內層迴圈。
- 用 `%` 運算子做整除判斷：`n % 3 == 0`。
- `while` 迴圈記得設定計數器變數，避免無限迴圈。
- `input()` 取得的輸入是字串，需用 `int()` 轉換後才能比較大小。

## 學習重點

**與爬蟲／ML 的關聯**

- **爬蟲**：抓取多頁資料時，你會大量使用 `for` 迴圈遍歷頁碼、`while` 迴圈等待頁面載入，以及 `break` 在找到目標時提早停止。
- **ML**：訓練模型時，`for epoch in range(n_epochs)` 是最核心的訓練迴圈；`continue` 可用來跳過損壞的資料批次；`break` 可在 early stopping 時中止訓練。
