# 第 2 週今日 Python 練習題：成績分析系統

## 題目說明

你是一位老師，需要快速分析一批學生成績。請撰寫一個程式，讓老師輸入若干筆成績後，自動計算平均分、最高分、最低分，並統計各等級（A/B/C/D/F）的人數。

這個練習直接對應到日後用 **Pandas** 分析資料表、用 **NumPy** 做統計，以及 ML 資料前處理中「逐筆掃描並分類」的核心思維。

---

## 技術要求

- **概念**：`for`、`while`、`range`、`break`、`continue`、累加器、計數器
- **需要的模組**：無（純 Python 內建）

---

## 具體任務

1. 請使用者輸入「要分析的成績數量」（`n`），並用 `range(n)` 控制輸入迴圈。
2. 在 `for` 迴圈中，每一輪請使用者輸入一個成績（0–100）：
   - 若輸入的不是數字（`try/except ValueError`），用 `continue` **跳過該輪**並提示「格式錯誤，略過此筆」，**不計入總數**。
   - 若輸入值超出 0–100，同樣用 `continue` 跳過並提示「成績需介於 0 到 100，略過此筆」。
   - 有效分數才加入列表 `scores`。
3. 全部輸入完畢後，**若 `scores` 為空**，印出「沒有有效成績可分析。」並結束。
4. 否則，計算並印出：
   - 平均分（保留兩位小數）
   - 最高分 / 最低分
   - 各等級人數（A：90–100，B：80–89，C：70–79，D：60–69，F：0–59）
5. **進階**：加入「提前結束」功能：使用者輸入 `q` 時，用 `break` 立刻停止輸入並分析已收集的成績。

## 提示

- 用一個空列表 `scores = []` 儲存有效成績，搭配 `scores.append(score)` 新增。
- `sum(scores) / len(scores)` 可算出平均；也可以試試 Python 內建的 `sum()`、`max()`、`min()`。
- 等級統計可用多個 `if/elif` 在同一個 `for score in scores` 迴圈裡完成。
- `f"{avg:.2f}"` 可將浮點數格式化為兩位小數。
- 進階的「輸入 `q` 提前結束」：在 `for` 迴圈一開始用 `if user_input == 'q': break`，記得把 `input()` 的結果先存成字串再判斷。

## 學習重點（跟爬蟲／ML 的關聯）

| 今天學的概念 | 爬蟲 / ML 的對應場景 |
|---|---|
| `for i in range(n)` | Pandas `iterrows()` 逐列處理；ML 訓練固定 epoch 次數 |
| `continue` 跳過無效資料 | 資料前處理：過濾空值、格式錯誤、超出範圍的異常值（outlier） |
| `break` 提前結束 | 爬蟲：找到目標資料後立即停止，避免多餘的請求 |
| `sum()` / `max()` / `min()` | NumPy `np.mean()` / `np.max()` / `np.min()` 的概念原型 |
| 等級分類計數器 | ML 中的「分箱（Binning）」技術：把連續數值分成離散類別 |

## 進階挑戰（選做）

1. **中位數**：成績排序後找中間值（不使用 `statistics` 模組，自己用索引計算）。
2. **直方圖**：用 `for` 迴圈和 `*` 運算子，印出橫向的 ASCII 長條圖，例如：
   ```
   A ████████ (8人)
   B █████    (5人)
   ```
3. **補考線**：若平均分低於 60，自動印出所有不及格學生的「編號」（1-indexed），提醒老師追蹤。
