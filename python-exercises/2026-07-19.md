# 第 2 週今日 Python 練習題：數字猜謎與倍數統計

## 題目說明

透過今天的練習，你將熟悉 Python 的迴圈語法。我們會寫兩個小程式：
1. 一個用 `while` 迴圈實作的猜數字遊戲
2. 一個用 `for` 迴圈統計特定範圍內倍數的程式

---

## 技術要求
- 概念：`for`、`while`、`range`、`break`、`continue`、條件判斷
- 需要的模組：`random`（內建，不需安裝）

---

## 具體任務

### 任務一：猜數字遊戲（while + break）

1. 用 `random.randint(1, 100)` 產生一個 1～100 的隨機整數作為答案
2. 用 `while True` 無限迴圈讓使用者持續猜數字
3. 每次猜測後給提示：「太大了！」、「太小了！」或「恭喜猜對了！」
4. 猜對後用 `break` 跳出迴圈，並顯示總共猜了幾次

### 任務二：倍數統計（for + continue）

1. 用 `for` 迴圈走訪 1～50 的所有整數（含）
2. 遇到同時是 3 和 5 的倍數（即 15 的倍數）時，用 `continue` 跳過不計入
3. 分別累加 3 的倍數總和、5 的倍數總和（15 的倍數不算）
4. 最後印出兩個總和

---

## 範例

```python
# 任務一範例輸出
請猜一個 1～100 的數字：50
太大了！
請猜一個 1～100 的數字：25
太小了！
請猜一個 1～100 的數字：37
恭喜猜對了！你總共猜了 3 次。

# 任務二範例輸出（部分）
3 的倍數總和（排除 15 的倍數）：208
5 的倍數總和（排除 15 的倍數）：105
```

---

## 提示

- `while True:` 搭配 `break` 是「直到條件成立才停止」的常見寫法
- `continue` 會跳過本次迴圈剩下的程式碼，直接進入下一輪
- 用一個計數器變數（如 `attempts = 0`）在每次猜測後 `+1`
- 判斷倍數用 `%`（取餘數），例如 `n % 3 == 0` 表示 n 是 3 的倍數
- `range(1, 51)` 會產生 1 到 50（包含 50）

## 學習重點（跟爬蟲／ML 的關聯）

| 語法 | 在爬蟲 / ML 的實際用途 |
|------|------------------------|
| `for` + `range` | 走訪多個網頁頁數（第 1 頁、第 2 頁…） |
| `while` + `break` | 等待 API 回應、重試失敗的請求直到成功 |
| `continue` | 跳過格式不正確的資料列（清理資料集） |
| 計數器變數 | 追蹤已爬取頁數、訓練 epoch 次數 |
