# 第 3 週今日 Python 練習題：溫度轉換工具

## 題目說明

建立一個溫度單位轉換工具，可以在攝氏（Celsius）、華氏（Fahrenheit）與克耳文（Kelvin）三種單位之間相互轉換。透過這道題練習如何定義函數、傳入參數、設定預設值，以及回傳計算結果。

## 技術要求
- 概念：`def`、`return`、位置參數、預設值參數（default parameter）、函數呼叫
- 需要的模組：無（純 Python）

## 具體任務

1. 定義函數 `celsius_to_fahrenheit(celsius)`，將攝氏轉換為華氏，公式：`F = C × 9/5 + 32`
2. 定義函數 `fahrenheit_to_celsius(fahrenheit)`，將華氏轉換為攝氏，公式：`C = (F - 32) × 5/9`
3. 定義函數 `celsius_to_kelvin(celsius)`，將攝氏轉換為克耳文，公式：`K = C + 273.15`
4. 定義一個整合函數 `convert_temperature(value, from_unit="C", to_unit="F")`：
   - `from_unit` 預設為 `"C"`（攝氏）
   - `to_unit` 預設為 `"F"`（華氏）
   - 支援的單位代號：`"C"`（攝氏）、`"F"`（華氏）、`"K"`（克耳文）
   - 若輸入不支援的單位，回傳 `None` 並印出錯誤訊息
5. 在主程式中呼叫 `convert_temperature()` 測試以下情境：
   - 100°C → 華氏
   - 32°F → 攝氏
   - 0°C → 克耳文
   - 使用預設值（不指定 `from_unit` 與 `to_unit`）轉換 25

## 範例

```python
# 輸入 / 輸出

result = convert_temperature(100, from_unit="C", to_unit="F")
print(result)   # 212.0

result = convert_temperature(32, from_unit="F", to_unit="C")
print(result)   # 0.0

result = convert_temperature(0, from_unit="C", to_unit="K")
print(result)   # 273.15

result = convert_temperature(25)   # 使用預設值：C → F
print(result)   # 77.0

result = convert_temperature(100, from_unit="X", to_unit="F")
# 輸出：不支援的單位：X
# print(result) → None
```

## 提示
- 先把每個轉換步驟拆成獨立的小函數，整合函數再呼叫它們，這樣邏輯最清晰。
- 若要支援 K → F 或 F → K，可先轉成攝氏當中繼，再轉到目標單位。
- `round(value, 2)` 可以讓輸出結果更整齊，避免浮點數誤差。
- 使用 `if/elif/else` 或字典（dict）判斷單位代號。
