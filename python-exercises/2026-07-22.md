# 第 3 週今日 Python 練習題：打造個人化 BMI 計算器

## 題目說明

透過撰寫多個函數，建立一個完整的 BMI（身體質量指數）計算工具。你將學習如何定義函數、傳入參數、設定預設值，以及回傳計算結果，這些技能是後續爬蟲與資料分析的核心基礎。

## 技術要求
- 概念：`def`、`return`、位置參數、關鍵字參數、預設參數值、函數組合
- 需要的模組：無（純 Python 即可）

## 具體任務

1. **定義 `calculate_bmi(weight, height)` 函數**
   - `weight`：體重（公斤）
   - `height`：身高（公尺）
   - 公式：BMI = weight / (height ** 2)
   - 回傳四捨五入到小數點第一位的 BMI 值

2. **定義 `bmi_category(bmi, standard="WHO")` 函數**
   - 預設採用 WHO 標準（`standard="WHO"`）
   - WHO 標準：
     - BMI < 18.5 → `"體重過輕"`
     - 18.5 ≤ BMI < 25.0 → `"正常範圍"`
     - 25.0 ≤ BMI < 30.0 → `"體重過重"`
     - BMI ≥ 30.0 → `"肥胖"`
   - 若 `standard="Asia"`，改用亞洲標準：
     - BMI < 18.5 → `"體重過輕"`
     - 18.5 ≤ BMI < 23.0 → `"正常範圍"`
     - 23.0 ≤ BMI < 27.5 → `"體重過重"`
     - BMI ≥ 27.5 → `"肥胖"`
   - 回傳對應的分類字串

3. **定義 `bmi_report(name, weight, height, standard="WHO")` 函數**
   - 呼叫前兩個函數來組合結果
   - 回傳格式化的報告字串（見範例）

4. **主程式**
   - 讓使用者用 `input()` 輸入姓名、體重與身高
   - 呼叫 `bmi_report()` 並印出結果
   - 同時印出 WHO 與亞洲標準的分類

## 範例

```python
# 函數呼叫範例
bmi = calculate_bmi(70, 1.75)
print(bmi)          # 輸出：22.9

category = bmi_category(22.9)
print(category)     # 輸出：正常範圍

category_asia = bmi_category(22.9, standard="Asia")
print(category_asia)  # 輸出：正常範圍

report = bmi_report("小明", 70, 1.75)
print(report)
# 輸出：
# === BMI 健康報告 ===
# 姓名：小明
# 體重：70 公斤 / 身高：1.75 公尺
# BMI 值：22.9
# WHO 標準：正常範圍
# 亞洲標準：正常範圍
# ====================
```

## 提示

- `round(數值, 1)` 可四捨五入到小數點第一位
- 函數可以呼叫其他函數：`bmi_report` 內部直接呼叫 `calculate_bmi` 和 `bmi_category`
- 預設參數要放在參數列的最後面：`def func(a, b, c="預設值")`
- 注意 `input()` 回傳的是字串，記得用 `float()` 轉換數字

## 進階挑戰（選做）

1. **加入輸入驗證**：若體重 ≤ 0 或身高 ≤ 0，讓函數印出警告並回傳 `None`

2. **批次處理**：定義 `batch_report(people)` 函數，接受一個 list，其中每個元素是 `(姓名, 體重, 身高)` 的 tuple，自動印出所有人的報告

   ```python
   people = [("小明", 70, 1.75), ("小華", 80, 1.70), ("小美", 55, 1.62)]
   batch_report(people)
   ```

3. **統計摘要**：在批次處理後，加一個 `summary(people)` 函數，印出這群人的平均 BMI、最高 BMI、最低 BMI
