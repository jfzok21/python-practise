# 第 3 週今日 Python 練習題：計算機函數工具箱

## 題目說明

你要建立一個「數學計算工具箱」，將常用的計算功能封裝成獨立的函數。透過這個練習，你將學會如何定義函數、使用參數（包括預設參數），以及讓函數回傳結果。

---

## 技術要求

- **概念**：`def`、`return`、位置參數、關鍵字參數、預設參數值、函數呼叫
- **需要的模組**：無（純 Python）

---

## 具體任務

1. **定義 `add(a, b)` 函數**：接收兩個數字，回傳它們的總和。

2. **定義 `power(base, exponent=2)` 函數**：  
   - 接收底數 `base` 和指數 `exponent`（預設值為 `2`）  
   - 回傳 `base` 的 `exponent` 次方  
   - 呼叫時若不傳入 `exponent`，預設計算平方

3. **定義 `bmi(weight, height)` 函數**：  
   - 接收體重（公斤）與身高（公尺）  
   - 計算 BMI = weight / height²  
   - 依據結果回傳對應的字串：  
     - BMI < 18.5 → `"過輕"`  
     - 18.5 ≤ BMI < 24 → `"正常"`  
     - 24 ≤ BMI < 27 → `"過重"`  
     - BMI ≥ 27 → `"肥胖"`

4. **定義 `greet(name, greeting="你好")` 函數**：  
   - 組合問候語並回傳字串，格式為：`"{greeting}，{name}！"`

5. **在主程式中呼叫以上四個函數，並 `print` 出結果**。

---

## 範例

```python
# 範例輸出

print(add(3, 5))             # 8
print(power(4))              # 16  （指數預設為 2）
print(power(2, 10))          # 1024
print(bmi(70, 1.75))         # 正常
print(bmi(45, 1.65))         # 過輕
print(greet("小明"))          # 你好，小明！
print(greet("世界", "哈囉"))  # 哈囉，世界！
```

---

## 提示

- 定義函數時，預設參數必須放在非預設參數**之後**（例如 `def power(base, exponent=2)` ✅，`def power(exponent=2, base)` ❌）。
- `return` 後面的值就是函數的「輸出」，記得每個分支都要有 `return`。
- BMI 計算中，身高記得是「公尺」，且要平方（`height ** 2`）。
- 使用關鍵字參數呼叫更清楚：`bmi(weight=70, height=1.75)`。

## 進階挑戰（選做）

1. **不定數量參數**：改寫 `add` 為 `add(*numbers)`，可接受任意多個數字並回傳總和。  
   範例：`add(1, 2, 3, 4, 5)` → `15`

2. **函數說明文件**：為每個函數加上 docstring（三引號說明），並用 `help(bmi)` 查看。

3. **遞迴挑戰**：用遞迴（函數呼叫自己）實作 `factorial(n)` 計算 n 的階乘。  
   範例：`factorial(5)` → `120`
