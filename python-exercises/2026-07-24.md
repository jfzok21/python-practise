# 第 3 週今日 Python 練習題：購物清單計算器

## 題目說明

你要為一間小型網路商店寫一個「購物金額計算模組」。透過定義多個函數，讓程式能計算單件商品小計、整張訂單總額，以及套用折扣後的最終金額。

## 技術要求
- 概念：`def`、`return`、位置參數、關鍵字參數、預設值參數
- 需要的模組：無（純 Python 標準語法）

## 具體任務

1. **定義 `item_subtotal(price, quantity=1)` 函數**
   - 接收單價 `price` 與數量 `quantity`（預設為 1）
   - 回傳小計金額（`price * quantity`）

2. **定義 `order_total(items)` 函數**
   - 接收一個 list，每個元素為 `(商品名, 單價, 數量)` 的 tuple
   - 在函數內呼叫 `item_subtotal()` 計算每項小計
   - 回傳所有小計加總（訂單總額）

3. **定義 `apply_discount(total, discount_rate=0.0)` 函數**
   - 接收總額 `total` 與折扣率 `discount_rate`（0.0 ~ 1.0，預設 0 表示不打折）
   - 回傳折扣後金額，**四捨五入到小數點後兩位**

4. **撰寫主程式**
   - 建立至少 3 筆商品的訂單清單
   - 依序呼叫上述三個函數，印出每項小計、訂單總額，以及打 9 折後的最終金額

## 範例

```python
# 輸入（訂單內容）
cart = [
    ("蘋果", 30, 3),
    ("香蕉", 15, 5),
    ("牛奶", 65, 2),
]

# 預期輸出
# 蘋果 x3 = $90
# 香蕉 x5 = $75
# 牛奶 x2 = $130
# 訂單總額：$295
# 九折後金額：$265.5
```

## 提示

- `def item_subtotal(price, quantity=1):` 中的 `quantity=1` 是預設值，呼叫時可省略不寫
- 用 f-string 印出金額，例如 `f"${amount}"`
- `round(number, 2)` 可將數字四捨五入到小數點後兩位
- 函數要先定義才能呼叫——確保 `item_subtotal` 寫在 `order_total` 之前

## 學習重點

函數是爬蟲與資料分析的基本建構單位：
- 爬蟲常把「請求一個頁面」、「解析 HTML」、「儲存資料」各自包成函數，讓主流程清晰易讀
- 資料處理也會把「清洗欄位」、「計算統計值」包成函數，方便在 Pandas 的 `apply()` 中重複使用
- 預設值參數讓函數更彈性，例如 `requests.get(url, timeout=10)` 就利用了這個機制
